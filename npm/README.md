# nexus-v (NPM Wrapper)

This is a lightweight Node.js wrapper for [Nexus-V](https://github.com/stillsystems/nexus-v), a modern VS Code extension scaffolder built in Go.

## Usage

You can run Nexus-V directly using `npx`:

```bash
npx nexus-v init
```

## Why use this?
While the native binary is faster and has zero dependencies, this NPM wrapper allows JavaScript developers to use Nexus-V without manually downloading binaries or adding them to their system PATH.
